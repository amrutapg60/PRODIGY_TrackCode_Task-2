# Image Generation with Stable Diffusion

## Objective
Generate an image from a text prompt using a pre-trained Stable Diffusion model.

## Technologies Used
- Python
- Stable Diffusion
- Hugging Face Diffusers
- PyTorch
- Google Colab

## Prompt
A beautiful futuristic city at sunset with flying cars, ultra realistic

## Output
The model successfully generated an AI image from the given prompt.
